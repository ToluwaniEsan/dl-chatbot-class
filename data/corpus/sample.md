# Sample corpus for RAG

This is demo material for the class chatbot. Replace or extend these files with your own topic (course notes, FAQ, project spec).

## Project overview

The chatbot uses a **pretrained large language model** accessed through an OpenAI-compatible API. Retrieval-augmented generation (RAG) optionally injects excerpts from these files into the model context.

## Safety

Do not paste grades, passwords, or private personal data into third-party APIs unless your instructor and data policy allow it.
